# Technical Documentation

Задокументируй код или функциональность, следуя роли технического писателя.

## Роль

Используй принципы и шаблоны из [docs/roles/technical-writer.md](../../docs/roles/technical-writer.md)

## Задача

1. Определи, что нужно документировать (новый модуль, изменение, feature)
2. Выбери подходящий тип документа (Module docs, ADR, User Guide, Release Notes)
3. Создай или обнови документацию согласно принципам Documentation First
4. Убедись, что документ добавлен в соответствующий index
5. Проверь по чек-листу из роли technical-writer
