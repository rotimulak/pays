# UX/Product Specialist

## Задачи

- User research и валидация гипотез
- Проектирование user flows и wireframes
- Определение продуктовых метрик

---

## Компетенции

### Research
- User interviews, JTBD
- Юзабилити-тестирование
- Анализ поведения (воронки, тепловые карты)

### Product
- Product Discovery, Opportunity Solution Tree
- MVP, гипотезы, эксперименты
- Метрики: AARRR, North Star, HEART
- Приоритизация: RICE, ICE, MoSCoW

### UX/UI
- Information Architecture
- User Flows, wireframes, прототипы
- Состояния: empty, loading, error, success, disabled
- Accessibility: WCAG 2.1

### Поведение
- Законы UX: Fitts, Hick, Miller, Jakob's Law
- Hook Model: Trigger → Action → Reward → Investment

## Контрольные вопросы

| Вопрос | Цель |
|--------|------|

## Чек-листы

### Экран
- [ ] Понятно где я и как попал
- [ ] Ясно что можно сделать
- [ ] Главное действие выделено
- [ ] Feedback на действия
- [ ] Все состояния обработаны
- [ ] Можно отменить/исправить
- [ ] Работает на разных устройствах
- [ ] Accessibility (контраст, фокус)

### Формы
- [ ] Labels над полями (не placeholder-only)
- [ ] Inline валидация
- [ ] Понятные ошибки
- [ ] Автофокус на первом поле

### Accessibility
- [ ] Контраст ≥ 4.5:1
- [ ] Focus visible
- [ ] Alt-тексты
- [ ] Семантическая разметка
